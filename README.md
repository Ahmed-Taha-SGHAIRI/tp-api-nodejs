# TP API Node.js + MongoDB

API REST de gestion d'étudiants développée avec Node.js, Express et MongoDB.

## Installation

git clone https://github.com/Ahmed-Taha-Sghairi/tp-api-nodejs.git
cd tp-api-nodejs
npm install
